# Test Method for Two-Axis Sensor Evaluation Kit

This guide introduces how to detect bending angle and stretch values of a one-axis sensor using Adafruit's Trinket M0.
For device connection and software operation, please refer to the Quick Start Guide.

## Required Materials
  - Two-axis sensor
  - Trinket M0 (Adafruit )
  - PC with Arduino IDE
  - USB Micro-B Cable

## Quick Start

  1) Connect the Trinket M0 and the Two Axis Sensor according to the wiring diagram in the Getting Started Guide.
  2) Set up Trinket M0 Arduino IDE.
  3) Run the code from the "Software_for_evaluation_kit" folder in this repository using the Arduino IDE.

## File Structure

  - /documentation: Getting Started Guide including wiring diagrams and setup instructions.
  - Release Page / Assets / Software ZIP Files (1 to 3)  : Please download the required software from the "Assets" section of the Release page.
